# Smart Lender - Loan Eligibility Prediction

## 📌 Project Overview

Smart Lender is a Machine Learning based web application that predicts whether a loan applicant is eligible for a loan based on the information provided by the user.

The application is developed using Python, Flask, Scikit-learn, HTML, and CSS.

---

## 🎯 Objective

The objective of this project is to help banks and financial institutions predict loan approval quickly and accurately using Machine Learning.

---

## 🛠️ Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS

---

## 📂 Dataset

The dataset contains the following attributes:

- Loan_ID
- Gender
- Married
- Dependents
- Education
- Self_Employed
- ApplicantIncome
- CoapplicantIncome
- LoanAmount
- Loan_Amount_Term
- Credit_History
- Property_Area
- Loan_Status

---

## 🤖 Machine Learning Model

Algorithm Used:

- Random Forest Classifier

## ▶️ How to Run

1. Install required libraries

```
pip install -r requirements.txt
```

2. Train the model

```
python train_model.py
```

3. Run Flask Application

```
python app.py
```

4. Open browser

```
http://127.0.0.1:5000
```
